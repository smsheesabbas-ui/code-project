# EPIC 1: Natural Language Query Engine

## Objective

Allow users to ask financial questions in natural language and receive clear, context-aware responses.

## Business Value

Removes dashboard dependency and increases engagement.
Turns financial analysis into conversation.

## Scope

* Chat interface
* Natural language understanding
* Query-to-data translation
* Contextual responses
* Plain-language formatting

## Functional Requirements

* Accept free-text financial queries
* Parse intent (e.g., revenue, expense, customer, forecast)
* Identify time references (e.g., last month, this week)
* Retrieve relevant data
* Return simple, insight-driven answers
* Maintain conversation context

## Non-Functional Requirements

* Response time < 3 seconds
* Plain language responses
* No accounting jargon
* Graceful fallback if question unclear
